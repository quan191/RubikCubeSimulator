# RubikCubeSimulator

Mô phỏng rubik 2x2 và 3x3

## Đề bài 

Mô phỏng rubik 2x2 và rubik 3x3 , ứng dụng lập trình ngôn ngữ hướng đối tượng bằng java , và tạo đồ họa bằng javafx .

## Phần mềm sử dụng  

* Java
* Javafx
* Javafx Scene Builder
